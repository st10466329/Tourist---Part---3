# **Explore South Africa Website**  
**Developer:** Kabelo Thamsanqa Rathete  
**Year:** 2025  

---

## **1. Project Overview**  
This website showcases the beauty and diversity of South Africa, providing a tourism guide to top destinations, services, and related content.  
The website includes interactive features like a rotating globe, embedded map, gallery lightbox, and fully functional forms.

---

## **2. Features Implemented**

### **HTML**
- Multi-page website (Home, About, Services, Enquiries, Contact)
- Structured headings and sections
- Accessible forms with proper labels
- Clean URLs and navigation

### **CSS**
- Full responsive design across all screen sizes
- Consistent color theme (green)
- Animations and transitions for interactive elements
- Layout improvements based on feedback from Part 2

### **JavaScript**
- Client-side form validation for **Enquiries** and **Contact** forms
- Error messages for missing fields
- Gallery lightbox for image viewing
- Dynamic map integration (using Google Maps or similar API)
- Content interaction improvements using JavaScript and CSS transitions

---

## **3. SEO Enhancements**

### **On-Page SEO**
- **Meta title** and **meta description** tags added for every page
- Relevant **meta keywords** for SEO optimisation
- **Alt text** added for all images
- Internal linking between pages for easy navigation
- Mobile-friendly layout and fast-loading pages

### **Off-Page SEO Preparation**
- **Social sharing buttons** to promote the website on social media
- Optimised for **local search** and targeted SEO

### **Technical SEO**
- **robots.txt** and **sitemap.xml** created
- Improved **page speed** for better user experience and SEO ranking
- Structured headings (H1, H2, etc.) used for SEO clarity

---

## **4. Forms and Validation**

### **Enquiries Form**
- Allows users to inquire about services, destinations, or bookings
- JavaScript validation ensures no empty fields are submitted
- Displays a response with information based on user input

### **Contact Form**
- Allows users to send general messages
- Validates name, email, phone number, and message type
- Collects information and generates a **formatted email** output for the user

---

## **5. Interactive Features**
- **Lightbox gallery** for displaying images
- **Rotating globe animation** on the homepage
- **Embedded Google Map** on the Contact page
- **Form validation** on Enquiries and Contact pages
- **Hover effects**, scroll animations, and dynamic UI interactions
